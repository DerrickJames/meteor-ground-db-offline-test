# cordova-plugin-legacy-whitelist

このプラグインは、コルドバ 4.0 のコルドバ 3.6 ホワイト リスト ポリシーを実装します。 安全な代わりに、コルドバ プラグインのホワイト リストを使用してください。

サポートされている: - cordova-android@4.0.0 - cordova-ios@4.0.0

## 使用法:

コルドバの以前のバージョンと同じように、 `<access>`タグを使用します。
