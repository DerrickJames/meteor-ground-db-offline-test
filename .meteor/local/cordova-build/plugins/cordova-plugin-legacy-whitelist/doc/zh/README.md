# cordova-plugin-legacy-whitelist

這個外掛程式實現科爾多瓦 4.0 科爾多瓦 3.6 白名單政策。 請改用科爾多瓦-外掛程式-白名單，因為它是更安全。

支援:-cordova-android@4.0.0-cordova-ios@4.0.0

## 使用方法:

使用`<access>`標記，就像以前版本的科爾多瓦。
