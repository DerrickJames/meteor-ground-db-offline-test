# cordova-plugin-legacy-whitelist

이 플러그인은 코르도바 4.0 코르도바 3.6 허용 정책을 구현합니다. 그것은 더 안전한 대신, 코르도바-플러그인-허용 사용 하십시오.

지원:-cordova-android@4.0.0-cordova-ios@4.0.0

## 사용법:

코르도바의 이전 버전에서와 마찬가지로 `< access >` 태그를 사용 합니다.
